八字命理分析系統 - 使用說明
====================================

快速開始
--------
1. 雙擊運行 start.bat（Windows）或執行 python server.py（其他系統）
2. 打開瀏覽器，訪問 http://localhost:5000
3. 填寫個人資料並點擊「開始分析」

功能說明
--------
✓ 八字四柱計算：自動計算年、月、日、時四柱
✓ 農曆轉換：顯示對應的農曆日期
✓ 生肖星座：顯示生肖和星座資訊
✓ 五行分析：分析命格中的五行分布
✓ 全球時區：支援多個時區的準確計算

輸入要求
--------
• 姓名：選填項目
• 性別：必填（男/女）
• 出生日期：必填（公曆日期）
• 出生時間：必填（24小時制）
• 時區：必填（選擇對應時區）

注意事項
--------
1. 確保輸入的出生時間準確，這會影響時柱的計算
2. 如果出生地與當前時區不同，請選擇正確的時區
3. 系統會優先使用 Bazi MCP 進行計算，如果 MCP 不可用，會自動切換到備用方法

技術支援
--------
• 需要 Node.js 才能使用 Bazi MCP 功能（可選）
• 如果沒有 Node.js，系統仍可正常運行，但使用備用計算方法
• 詳細技術說明請參閱 README.md

常見問題
--------
Q: 為什麼計算結果與其他系統不同？
A: 八字計算涉及多種算法，不同系統可能採用不同的計算方法。
   本系統使用 Bazi MCP 提供更準確的計算。

Q: 如何提高計算準確度？
A: 1. 確保 Node.js 已安裝並運行 Bazi MCP
   2. 輸入準確的出生日期和時間
   3. 選擇正確的時區

Q: 系統無法啟動怎麼辦？
A: 1. 確認 Python 3.7+ 已安裝
   2. 執行 pip install -r requirements.txt
   3. 檢查端口 5000 是否被占用
